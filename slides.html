How to deploy your tech website portfolio for FREE
  • Create a repo and place your files into it
  • Name the repo: "your-username.github.io"
    *Has to be exactly the same as your username or it won't work
  • Settings -> Pages -> Deploy from a branch -> Select branch you want to deploy *usually main
